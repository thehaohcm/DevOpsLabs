Create a tar file:
# tar -t [filename].tar /path

Extract a tar file:
# tar xf [filenam].tar

Create a compress file:
# tar cvf[X] [filename].tar.[Y] /path1/ /path2/

Extract a compress file:
# tar xvf[X] [filename].tar.[Y] [--C /extracted path/]

X = x => Y = gz
X = j => Y = bz/tbz (bzip2)
X = J => Y = xz
